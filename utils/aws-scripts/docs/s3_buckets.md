# Describe S3 Bucket

This script retrieve information about S3 Buckets in all regions. Here's a brief overview of what the script does:

* Defines a function called `list_s3_buckets_by_region` that takes a single argument, region_name. This function uses the boto3 library to create a client for the EC2 service in the specified region, and then uses the `list_s3_buckets` method to retrieve information about all S3 Buckets in that region.
* The function then prints out the total number of instances found in the region, and loops through each instance to print out its name, ID, and instance type.
* If an error occurs while trying to retrieve the instance information, the function catches the exception and prints out an error message.
* Finally, the function returns a summary of the total number of instances in all the regions.
